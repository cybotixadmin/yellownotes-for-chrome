edit the template for your yellownotes
This is the default template for all notes created by you. You can change this template at any time and it will affect all news notes you create

  • height
    • changing this will only affect all new note you create. It will not affect the height of notes you have already created
  • width
    • changing this will only affect all new note you create. It will not affect the width of notes you have already created
  • background color
    • changing this color will affect all note you have created
  • banner image
    • changing this image will affect you your notes, including all notes you have already created
