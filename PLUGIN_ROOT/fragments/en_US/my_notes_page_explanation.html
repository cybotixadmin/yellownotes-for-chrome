Created
This column contains the date and time when the note was created. You can sort the notes by creation date by clicking ▲ (or ▼) on the column header.
This information can not be modified
Changed
This column contains the date and time when the note was last modified. You can sort the notes by the last modification date by clicking ▲ (or ▼) in the column header.
When the note is changed, the date and time will be updated.
Type
This column contains the type of the note. The note type can be either "webframe", "capture_note" or "yellownote".
The note type can not be changed
Act
This column contains the active/disabled status of the note. You can filter the notes by their status by selecting "active" or "inactive" from the dropdown menu.
Modify the notes active/inactive status by (un)ticking the checkbox. The change is immediate and there is not need to save the change.
Location
This column shows the web address the note is attached to. You can sort the notes alphabetically by this address by clicking ▲ (or ▼) in the column header.
The address can be updated by editing the cell directly. Any changes must be saved by clicking the save (floppy disk icon) icon at the end of the line.
Selected
This column shows the text of the note is attached to, if any. This is the text slected on a web page when the note was origianlly created. You can sort the notes alphabetically by this text by clicking ▲ (or ▼) in the column header.
To modify selection text, edit the contents of the cell directly. Any changes must be saved by clicking the save (floppy disk icon) icon at the end of the line changes to this text can make the note no longer match the original text. This will affect how the note is placed on the target page
Message
This column shows the message in the yellow note. You can sort the notes alphabetically by this message by clicking ▲ (or ▼) in the column header.
To modify message text, edit the contents of the cell directly. Any changes must be saved by clicking the save (floppy disk icon) icon at the end of the line
Action
This column shows action that can be take on the note

The delete icon. (the trash can icon) Deletion takes place immediately and there is no prompting for confirmation.

The go-to icon. (arrow to document icon) This icons contains a link to the page the note is attached to. Clicking the icon will open the page in a new tab. And show the note in its placed location This link is suitable for sharing the note with others. Note: only notes that are attached to a feed are sharable.
clicking the save icon. saved by clicking the save (floppy disk icon)
Feed
This column shows the feed (distribution list) the note is attached to, if any. A note that is not attached to any feed is private to you.
To modify which feed the note is attached to, select a new feed from the drop-down list contained in the cell. To remove the note for any feed, simply select the empty option on top of the drop-down list. This makes the note private to you. Changes are immediate, and there is no need to save the change by clicking the save icon.
